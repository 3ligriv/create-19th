# Create 19th
## A Minecraft modpack evolving around Create in the end of the 19th century

### Description
A modpack for playing on a server with an aesthetic and RP focus.

### Core mods

- Create & many addons
- Terralith & Tectonic
- Farmer's Delight & Spice of Life Onion
- Artifacts & Better Archeology
- Many decoration mods: Chipped, Macaw' suite, Unusual Furniture,...
- Lightman's Currency & Simple Voice Chat
- And many QoL mods...

### Create addons

- Create Deco
- Create Railways Navigator
- Create Slice & Dice
- Create: Bells & Whistles
- Create: Central Kitchen
- Create: Power Loader
- Create: things and misc
- Create: Design n' Decor
